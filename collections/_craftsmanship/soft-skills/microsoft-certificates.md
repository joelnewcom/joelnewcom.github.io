# Points
Every correct answer/choice will give points.
Worst thing to do is to not answer a question. There are no "minus points". 

# Break
If you take a break, every question you opened will be set to "answered" and you cannot change them.

# Solution questions:
There are problem described and the question is, if the presented solution works to solve the problem.
You can not go back to previous answered questions. 
I can be, that all the solutions will not work or work. 

Don't think you HAVE to answer yes, because you need a solution for this problem. There might be all wrong solutions.
